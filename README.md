# Trader Behavior Insights

## Objective
Analyze relationship between Bitcoin market sentiment (Fear/Greed) and trader performance.

## Dataset
- Fear & Greed Index
- Historical Trader Data

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Key Analysis
- Profit vs Sentiment
- Win Rate Analysis
- Trade Size Behavior
- Clustering of Traders

## Key Insights
- Traders perform better during Greed periods
- Larger trades observed during Greed
- Fear leads to lower profitability

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run Jupyter Notebook
